The work on Optical Cense progressed and we made the Version 2 for this project. I was appointed a Research Assistant for this work.

In the previous version, we had used ENDEC which worked at 115,2 Kbaud. Now according to the requirement, we needed high transfer rates and so we replcaed endec with MCP2120. The MCP works at 312.5 Kbaud which is almost twice faster than the previous IC.

The main problem we faced was the our board was working on 3.3 V and this IC requied 5 V to work. So we needed to design power sources using buck converters.
