MathFact Game

During my 6 month internship at a startup company called Playpower Labs, I learnt to design games and code them in flash using AS3. One of the games which I made and field tested in schools was MathFact.

MathFact is a simple math game which test the mathematical skills in addtion, subtraction, multiplication and divison. The game has feedback and achievement system, fun graphics and intuitive UI, teacher dashboard to view student progress and student leaderboard to view top performers

The game was made using AS3 programming language on Pure MVC framework. Apart from coding, I was involved in making the gameplay, design and field testing. Link to the game -MathFact Game
